# Personal Blogging Website – README

## Project Description

This project is a personal blogging website created using **HTML and CSS** for the course **CS106 – Fundamentals of Web Design**.
The purpose of this project is to demonstrate the use of CSS styling techniques, layout systems, and responsive design while building a multi-page website.

The website follows the concept of **separation of content and presentation**, where HTML is used for structure and CSS is used for styling.

---

## Pages in the Website

### 1. Homepage (index.html)

The homepage introduces the blog and contains:

* Blog title
* Navigation bar
* Short introduction
* Preview of blog posts

### 2. Blog Page (blog.html)

This page contains **five blog posts**.
Each post includes:

* Title
* Author name
* Date
* Content

### 3. About Page (about.html)

The about page provides information about the author and the purpose of the blog.

### 4. Contact Page (contact.html)

This page contains a **contact form** where users can enter their name, email, and message.

---

## CSS Features Implemented

### CSS Selectors

Different selectors are used including:

* Element selectors
* Class selectors
* ID selectors
* Group selectors
* Combinator selectors

### Box Model

The CSS box model is demonstrated using:

* Margin
* Padding
* Borders
* Width and height

### Navigation Bar

A horizontal navigation bar with styled links and hover effects is included on all pages.

### Layout System

Flexbox is used to arrange blog posts and page content.

### Pseudo-Classes

Interactive pseudo-classes used include:

* :hover
* :focus
* :nth-child

### Responsive Design

Media queries are used to make the website responsive and adaptable to smaller screen sizes.

### Form Styling

The contact form fields and submit button are styled using CSS.
